# **App Name**: Vocabify

## Core Features:

- Grammar Puzzle Engine: Interactive grammar challenges presented as puzzles. Includes rapid-fire grammar correction and sentence fixing under constraints.
- Vocabulary Guessing Game: Users guess words using visual clues, hints, and partial correctness feedback, similar to Wordle mechanics.
- Sentence Flow Lab: Users reorder sentences or clauses to form a logical paragraph, enhancing coherence and logical sequencing skills.
- Error Hunt Mode: Users scan passages to find grammar, usage, or vocabulary errors within a time limit, building attention to detail. A tool will be used to suggest types of common error for specific users.
- Story Mode: Grammar and vocabulary challenges are embedded into a continuous story for contextual learning and higher engagement.
- Grammar Synonym & Usage Challenge: Users choose correct synonyms or word replacements based on sentence context, testing depth of understanding.
- Rapid-Fire Challenge Mode: Timed mini-challenges with increasing difficulty to improve speed of comprehension and automaticity in language use.

## Style Guidelines:

- Primary color: Vibrant blue (#42A5F5), representing intelligence and clarity.
- Background color: Light blue (#E3F2FD), creating a calm and focused environment.
- Accent color: Energetic orange (#FFB74D) for highlighting important elements and calls to action.
- Body text and Headlines: 'PT Sans' (sans-serif) to balance a modern feel and a little warmth/personality; suitable for both headlines and body text.
- Code font: 'Source Code Pro' (monospace) for displaying code snippets (should errors contain code).
- Simple, clear icons to represent different grammar concepts and challenge types.
- Subtle animations for correct answers and level transitions, providing positive feedback.
